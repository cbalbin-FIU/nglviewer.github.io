Class StlWriter

Create an STL File from a surface Object (e.g. for 3D printing)

example

molsurf = new MolecularSurface(structure) surf = molsurf.getSurface({type: 'av', probeRadius: 1.4}) stl = new StlWriter(surf) stl.download('myFileName')
